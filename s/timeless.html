What is this?

Notable works I've stumbled upon, ordered by the year (and place) of birth of their creator. You can derive a lot of information from this simple binomial, the zeitgeist is inevitable. Lifetime work-in-progress. You may also like underrated.

Why?

I wanted to read something like this - an information flood is drowning our times - but I couldn’t find it elsewhere. So, as most projects, it started while «scratching my own itch» and before becoming a website it lived as a markdown file.

Making sense of chaos is a sisyphean task but I see it as a way of using my procrastination positively, I’m an analist by heart.

How do you curate it?

I cherry-pick relevant stuff and list them carefully. It’s certainly affected by my biases (I’m aware of these).

How often do you update this section?

Only when relevant. New listings are easy to spot.

Can I contribute?

Open an issue or pull request the repository with relevant links, corrections or suggestions. I’m curating only a few pages but I accept pull requests creating/curating more - if you can handle the burden of using raw HTML. For now, this is my filter to accept them.

Feeling grateful?

Shop something through my Amazon affiliate link (you pay exactly the same but it keeps my motivation high), donate what you think is fair or just say hi. You can also sponsor this page. I don’t want to publish intrusive ads neither data farm with yet another stupid login.

Colophon

Hand coded - fully functional without JS - and statically hosted at Netlify via a Github repository. Tufte CSS (adapted) with native darkmode. No images, no tracking (GDPR friendly analytics with Goatcounter) and no bloat. Ok, it uses non-standard fonts but web design is mostly typography.
