Sponsoring. How does it work?

  • Only the main page is open for sponsors - for now.
    Non-negotiable: only one sponsor per page.

  • Actual price: 100€ quarterly.

    If the spot is taken, you may propose a price «raise» to get it next quarter. A couple of weeks before the quarter’s end, the incubent will be informed of the highest raise - if any. If he doesn’t «call» the price, his sponsoring will be terminated. The actual price will be updated here.
  • The sponsor will get one colored text link (adblock resistant - no JS) on the top of the main page and on the quarterly newsletter. 140 UTF characters allowed. Quarterly - as a call for attention - you may change the link color.
  • Past traffic - that doesn’t use adblock - is showed here. Most traffic is direct. Most people found Slower News on Hacker News.
  • Paid by Paypal or Crypto. Don’t expect receipts. Like Slower News itself, the process is analogic and amateur. Hopefully trustable.
  • Even if something is less palatable... say hi, maybe we can work it out. Above all, I want everybody happy. Sponsors and readers.

WTF is Slower News?

These are, somehow, curated news extracting relevant trends and some edge cases for borderline nerds, that don’t want to miss out, nor spend a shit-ton of time filtering trivia.

Why?

I wanted to read something like this - an information flood is drowning our times - but I couldn’t find it elsewhere. So, as most projects, it started while «scratching my own itch» and before becoming a website it lived as a markdown file.

Making sense of chaos is a sisyphean task but I see it as a way of using my procrastination positively.

How do you curate articles?

I cherry-pick articles that dissect trends or unveil lesser known trends and interesting edge cases, and are relevant, at least, for some months. It’s certainly affected by my biases (I’m aware of these) and most articles were on Hacker News. The articles’ listing order is not random.

How often is Slower News updated?

Only when relevant. Visit every month and «you’re good to go». New articles are easy to spot but there is RSS and a newsletter also.

Can I contribute?

Open an issue or pull request the repository with relevant links, corrections or suggestions. I’m curating only a few pages (this, one about Portugal and Timeless) but I accept pull requests creating/curating more - if you can handle the burden of using raw HTML. For now, this is my filter to accept them.

Colophon

Hand coded - fully functional without JS - and statically hosted at Netlify via a Github repository. Tufte CSS (adapted) with native darkmode. No images, no tracking (GDPR friendly analytics with Goatcounter) and no bloat. Ok, it uses non-standard fonts but web design is mostly typography.
